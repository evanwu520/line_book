線上圖書借閱系統 — Java + Spring Boot 專案開發考題 背景 某市的圖書館希望開發一個線上系統,讓使用者可以查詢各館的館藏並進行借閱與還書, 請你以 Java + Spring Boot 進行實作,並以真實專案的開發方式撰寫程式碼。 完成後請將專案上傳至 GitHub(公開或提供訪問權限),並確保程式碼可編譯、可運 行。 請盡可能考慮到所有可能的情境。 功能需求 1. 會員管理 • 角色: o 館員(Librarian) o 一般用戶(Member) • 功能: o 使用者可註冊、登入。 o 建立館員帳號時,需與圖書館另一系統驗證(模擬 API 呼叫): § GET Header: { "Authorization": "todo" } URL: https://todo.com.tw o 權限控制: § 館員可管理書籍資料。 § 一般用戶不可新增/修改書籍資料。 2. 書籍管理 • 書籍資訊: o 書名 o 作者 o 出版年份 o 類型(圖書、書籍) • 功能: o 館員可新增書籍至系統(支援多館館藏與同館多副本)。 o 使用者可依書名、作者、年份搜尋書籍。 o 搜尋結果需顯示: § 書籍基本資訊 § 各館館藏數量(可用數量)。 3. 借閱與還書 • 借閱規則: o 每位使用者同時最多可借: § 圖書:5 本 § 書籍:10 本 o 借閱期限:1 個月。 • 功能: o 借書與還書功能。 o 到期前 5 天需發送通知(實作可用 System.out.println 模擬訊息發送)。 o 借閱數量或期限超過規則時需提示錯誤。
